## Tips about exercises
- Using `object.info`.
```js
let classes = [
    {
        classInfo: {
            name: 'A',
            capacity: 40,
            type: 'math'
        },
        [
            //Students
        ]
    }
]
```

- Using Uniq `ID`, For direct and convenient access.
```js
let classes = [
    {
        classInfo: {
            name: 'A',
            capacity: 40,
            type: 'math',
            classID: 1001 // ID
        },
        [
            //Students
        ]
    }
]
```
